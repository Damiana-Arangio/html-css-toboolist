<h1 align="center">Toboolist</h1>

Pagina HTML/CSS che riproduce il layout di una semplice Todo List (“Toboolist”),
con attenzione alla suddivisione in macroaree e alla riutilizzabilità delle regole CSS.

## Obiettivo

- Ricreare il layout fornito della Todo List utilizzando HTML e CSS
- Individuare le macroaree del layout (header, contenuto principale, liste di task, ecc.)
- Analizzare gli elementi in comune tra le varie sezioni e centralizzare il più possibile le regole CSS,
  in modo da avere un codice meno ripetitivo e più mantenibile
